# Pokedex App

Ứng dụng Android tra cứu thông tin Pokemon sử dụng PokeAPI. Được xây dựng với Jetpack Compose và Dagger Hilt.

## Công nghệ sử dụng
* **Ngôn ngữ:** Kotlin
* **UI:** Jetpack Compose, Material 3
* **Dependency Injection:** Dagger Hilt
* **Navigation:** Jetpack Navigation

## Hướng dẫn cài đặt và chạy thử
1. **Clone repository:**
   ```bash
   git clone https://github.com/username/Pokedex.git
   ```
2. **Mở dự án:**
   Mở **Android Studio**, chọn *Open* và trỏ đến thư mục dự án đã clone.
3. **Đồng bộ Gradle:**
   Đợi Android Studio đồng bộ các thư viện qua Gradle (Gradle Sync) hoàn tất.
4. **Chạy ứng dụng:**
   Kết nối thiết bị Android vật lý hoặc mở Emulator, sau đó nhấn nút **Run** (Shift + F10).
